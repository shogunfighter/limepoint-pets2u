### Notable packages
Typescript and helper

```
$ npm install typescript ts-node --save-dev
```

Jest and helper
```
$ npm install --save-dev jest @types/jest ts-jest jest-cli jest-coverage
```

CLI helper

```
$ npm install yargs -D
```



### To get started

Clone and install repo
```
$ npm i
```

### NPM Scripts
